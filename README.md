# 🚀 AlgoRythmX

AlgoRythmX is an advanced interactive Algorithm Visualizer built using HTML, CSS, and JavaScript. It provides real-time, step-by-step visualization of algorithms from Data Structures, Operating Systems, Graph Theory, Memory Management, Disk Scheduling, and Computer Graphics.

---

## 🔥 Features

### 📊 Sorting Algorithms
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort (with recursion tree)
- Quick Sort (pivot visualization)
- Heap Sort (heap tree + extraction view)

---

### 🔍 Searching Algorithms
- Linear Search
- Binary Search (with range highlighting)

---

### ⚙️ CPU Scheduling
- FCFS (First Come First Serve)
- SJF (Shortest Job First)
- Round Robin (with time quantum)

✔ Gantt Chart visualization  
✔ Waiting & Turnaround time calculation  
✔ Ready Queue animation  

---

### 💽 Disk Scheduling
- FCFS
- SSTF
- SCAN (Elevator Algorithm)

✔ Disk head movement animation  
✔ Seek distance calculation  

---

### 🧠 Memory Management

#### 🔹 Page Replacement
- FIFO
- LRU
- Optimal

✔ Frame-by-frame visualization  
✔ Page fault counter  
✔ Timeline view  

#### 🔹 Memory Allocation
- First Fit
- Best Fit
- Worst Fit

✔ Block allocation visualization  
✔ Internal fragmentation display  

---

### 🌐 Graph Algorithms
- BFS (Queue visualization)
- DFS (Stack visualization)
- Dijkstra’s Algorithm

✔ Graph traversal animation  
✔ Distance table updates  
✔ Edge relaxation highlighting  

---

### 🎨 Computer Graphics (CGIP)
- DDA Line Drawing
- Bresenham Line Drawing
- Midpoint Circle Algorithm

✔ Pixel grid visualization  
✔ Coordinate system  
✔ Step-by-step plotting

## 📌 Project Description

AlgoRythmX is designed to simplify complex computer science concepts through visual learning. It allows users to observe algorithm execution in real time, making it easier to understand logic, flow, and performance.

This project demonstrates strong understanding of:
- Data Structures & Algorithms
- Operating Systems concepts
- Graph Theory
- Computer Graphics
- Frontend Development
